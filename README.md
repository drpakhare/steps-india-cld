# STEPS-INDIA · Systems Evidence for Peer Support

**Live site:** https://drpakhare.github.io/steps-india-cld

## Structure
```
steps-india-cld/
├── index.html          ← Overview landing page
├── cld.html            ← Animated CLD viewer (HTN · DM · eCES)
├── docs/
│   ├── Module1_Introduction_PSG_Framework.docx
│   ├── Module2_Evidence_Base_Full_eCES.docx
│   └── Module3_CLDs_Loop_Analysis.docx
└── audio/
    ├── htn/  step_01.mp3 … step_12.mp3   ← ElevenLabs
    ├── dm/   step_01.mp3 … step_09.mp3   ← Descript
    └── eces/ (browser TTS, no files needed)
```

## Deployment (GitHub Pages)
1. Create repo `steps-india-cld` (Public)
2. Upload all files maintaining the folder structure above
3. Settings → Pages → Branch: main → Save
4. Upload MP3 files to `audio/htn/` and `audio/dm/`
5. Live at `https://YOUR-USERNAME.github.io/steps-india-cld`
